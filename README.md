# 💻 Example ROS2 Workflow

```bash
# Source ROS2
source /opt/ros/humble/setup.bash

# Create Workspace
mkdir -p ~/ros2_ws/src

# Build Workspace
cd ~/ros2_ws
colcon build

# Source Workspace
source install/setup.bash

# Launch Simulation
ros2 launch my_robot simulation.launch.py
```
